## 安装angularjs
- bower install angular --save
>-一般采用npm安装后台服务，用bower装前台，但是npm既可以装前端又可以装后台的，但是bower不能安装后台的
>>>>>>注意所有配置文件例如 .bowerrc 或者package.json等文件就是配置文件，不可以在里面写注释等乱七八糟的东西；

## 服务
- provider
//provider可以配置
app.provider("名字",function(){
    this.$get=function(){
        return {
            xx:fun..
        }
    }
})

- factory
//工作中最常用
app.factory("名字"function(){
  return {
     xx:fun..
       }
});

- service
//service的回调函数是factory return的对象的构造函数
app.service("名字",function(){
    this.xx=function....


});

- value
app.value("名字",{
    xx:function(){},
    yy:fun....
});
