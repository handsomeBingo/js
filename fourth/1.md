## 安装angular
- npm node package manager
## bower管理前端包的
```
npm install bower -g
```
## bower初始bower.json文件
- 用来记录依赖,文件夹目录不能大写 中文 特殊字符
```
bower init
```
## 安装angularjs
- 默认安装到bower_components文件夹下
```
bower install angular --save
```
## 指定文件的安装目录
- 创建名为.bowerrc的文件
- 配置指定目录{"directory":"./lib"}
- 再次进行安装

> 一般采用npm来安装后台包 用bower来安装前端包,所有配置文件都不能增加注释
